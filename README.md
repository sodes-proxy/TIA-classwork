# TIA-classwork
classwork of TIA class, open cv with python

## Install first
In order to make this work please install the necessary modules with the following commands:

- opencv
```
pip install opencv-python
```
- numpy
```
pip install numpy
```
- matplotlib
```
pip install matplotlib
```
